# Field notes

This is where we log the practice. Each week, carry one of the [seven frequencies](../README.md#the-seven-frequencies) into the world and leave a short note here.

Notes are not graded or ranked. They are a record of showing up.

## How to add one

Open a pull request that adds a file named `YYYY-MM-DD-yourhandle.md`. An alias is fine. Keep it short and honest.

Use this shape:

```
Date:
Frequency: (which of the seven)
What I did:
What it changed:
```

That's all. Return often.
